# UI Polish Cycles — Диплом

Цель: довести визуал до "не стыдно показать" через итеративные циклы скрин→фикс.
Правило: +1 к счётчику за каждый полный цикл (скрин → просмотр → орда агентов → фикс → проверка).

## Текущий счётчик: 0

## Базовые требования (от пользователя)
1. Шрифт — плавный, подключен локально (не Google Fonts CDN)
2. Фон — рабочий dark baseline, не сломанный
3. Текст — русский язык в интерфейсе
4. Масштаб — мобила 320-768px приоритет
